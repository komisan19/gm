# gm

create gist file

ex. Recommended for uploading notes.

# Set GitHub Token

```
export GITHUB_TOKEN=<your github token>
```

# Usage

```
gm -f sample.md
```

- option

```
gm -d this is sample -f sample.md
```

# Installation

```
go install github.com/komisan19/gm@latest
```
